# Payslip Generator

A professional, dynamic payslip generator built with React and Vite.

## Features

- **Dynamic Editor**: Easily update company, employee, and pay period details.
- **Flexible Earnings & Deductions**: Add multiple rows for earnings and deductions with automatic calculations.
- **Real-time Preview**: See the payslip update instantly as you type.
- **Print & PDF Export**: Built-in functionality to print or save the payslip as a PDF.
- **Professional Design**: Clean, modern layout suitable for official use.

## How to Run

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Click the link shown in the terminal (usually `http://localhost:5173`).

## Building for Production

To create a production-ready build:

```bash
npm run build
```

The built files will be in the `dist` directory.
